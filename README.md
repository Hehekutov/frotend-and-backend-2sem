# Практики 13-17

В этом репозитории находится мой проект по практическим работам 13-17 по дисциплине "Фронтенд и бэкенд разработка".
Проект выполнил студент ЭФБО-18-24 Кудояров Денис.

Основная работа находится в папке `prac13-17`. В проекте сделано приложение "Заметки", которое постепенно дорабатывалось от обычного веб-приложения до PWA с push-уведомлениями и напоминаниями.

## Что реализовано

- добавление и удаление заметок;
- хранение заметок в `localStorage`;
- офлайн-работа через `Service Worker`;
- `manifest.json` для установки приложения как PWA;
- работа через `HTTPS` на локальном сервере;
- архитектура `App Shell` с отдельной подгрузкой контента;
- push-уведомления;
- отправка событий в реальном времени через `WebSocket`;
- заметки с напоминанием по дате и времени;
- кнопка отложить напоминание на 5 минут.

## Какие технологии использовались

- `HTML` - структура страниц;
- `CSS` - оформление интерфейса;
- `JavaScript` - логика на клиенте;
- `Node.js` - запуск серверной части;
- `Express` - сервер и раздача статических файлов;
- `Socket.IO` - обмен событиями между клиентом и сервером в реальном времени;
- `web-push` - отправка push-уведомлений;
- `Service Worker` - кэширование и офлайн-режим;
- `Web App Manifest` - оформление PWA;
- `localStorage` - локальное хранение заметок в браузере.

## Структура проекта

- `prac13-17/index.html` - основная страница приложения;
- `prac13-17/content/` - отдельные части интерфейса;
- `prac13-17/app.js` - клиентская логика;
- `prac13-17/sw.js` - service worker;
- `prac13-17/server.js` - сервер для практик 15-17;
- `prac13-17/manifest.json` - настройки PWA;
- `prac13-17/vapid-keys.json` - VAPID-ключи для push-уведомлений;
- `prac13-17/localhost.pem` и `prac13-17/localhost-key.pem` - сертификаты для локального HTTPS.

## Как запустить

1. Установить зависимости:

```bash
npm install
```

2. Заполнить файл `prac13-17/vapid-keys.json` своими ключами.

3. Запустить проект:

```bash
npm run prac13-17:start
```

4. Открыть адрес, который появится в консоли.

Если сертификаты `localhost.pem` и `localhost-key.pem` лежат в папке `prac13-17`, приложение будет доступно по `HTTPS`.
